# 안전한 의약품 소비 가이드 시스템
 
 ## 1. 팀 소개 아프조
 - 팀원 : 양우령, 안소연,황태림, 임원경, 김명석 

 ## 2. 프로젝트 소개
 ### 1) 배경
 - 의약품 소비량과 1인당 의료기관 방문 횟수가 늘어나고 있다. 
 - 그에따른 각종 의료사고와 의약품 오남용으로 인한 피해사례가 늘어나고 있다.
 - 데이터 분석적 접근을 하여 이러한 피해를 사전에 방지하려고 한다.
